DateTime — class, package sb.datetime
An immutable DateTime object. There a multiple ways to create a DateTime.
DateTime(timeZone:sb.datetime:TimeZone, year:int, month:int, day:int, hour:int, minute:int, second:int) — Constructor, class sb.datetime.DateTime
Creates a new DateTime There are several ways to create a DateTime by using the creational subclasses; however, all creational subclasses ultimately use this constructor. This constructor also supports rollover values the same way that the Flex Date object does.
